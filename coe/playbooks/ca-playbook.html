Cloud Adoption Playbook

Introduction

Modern IT moves quickly- agile delivery of features makes the people that use systems happy and swift patching protects systems and data. To get these results, your agency will need to think, work and buy technology differently. Cloud adoption, if done correctly, can be a rallying point to modernize processes, security and workforce skills.

The Cloud Adoption COE assists with the analysis of current systems and applications to provide recommendations for planning migration to the cloud, including balancing tactical “lift and shift” imperatives with more strategic “uplift and shift” possibilities. Using an agile and flexible framework the CoE will accelerate adoption across the executive branch enterprise.
